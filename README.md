# School AI Solver

Интеллектуальное приложение для решения школьных задач по различным предметам. Поддерживает вввод условия задачи текстом и загрузку фотографий. Результаты сохраняются в историю для последующего анализа.

## Возможности

- ✓ Решение задач по различным школьным предметам
- ✓ Ввод условия задачи текстом
- ✓ Загрузка фотографий задач (PNG, JPG, JPEG, WEBP)
- ✓ Пошаговое решение с формулами и объяснением
- ✓ Демонстрационные геометрические рисунки
- ✓ Сохранение решений в историю (JSON)
- ✓ Современный темный интерфейс с розовыми акцентами
- ✓ Многопоточность для предотвращения зависания интерфейса

## Будущие возможности

- Подключение OpenAI API для настоящего решения задач
- Автоматическое распознавание текста с фотографий
- Автоматическое определение предмета
- Специализированная обработка геометрических задач с построением чертежей
- Расширенная матричная обработка физических и химических задач
- Поддержка новых предметов и субъектов

## Установка

### Требования

- Python 3.12 или выше
- pip (менеджер пакетов Python)

### Системные зависимости (Linux)

Для Ubuntu и Debian установите библиотеки Qt:

```bash
sudo apt update
sudo apt install libxcb-cursor0 libxkbcommon-x11-0 libxcb-xinerama0 libxcb-icccm4 libxcb-image0 libxcb-keysyms1 libxcb-render-util0
```

### Подготовка окружения

1. Клонируйте репозиторий или перейдите в директорию проекта

2. Создайте виртуальное окружение:

```bash
python3 -m venv .venv
```

3. Активируйте виртуальное окружение:

```bash
source .venv/bin/activate
```

4. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Запуск приложения

```bash
python main.py
```

## Архитектура приложения

Приложение использует профессиональную многоуровневую архитектуру:

- **UI слой** (`app/ui/`) - отвечает за отображение интерфейса
- **Контроллеры** (`app/controllers/`) - связывают интерфейс и логику, обрабатывают события
- **Сервисы** (`app/services/`) - содержат бизнес-логику (ИИ, история, геометрия)
- **Модели** (`app/models/`) - структуры данных задач и решений
- **Утилиты** (`app/utils/`) - вспомогательные функции

## Дизайн интерфейса

Приложение использует современный темный дизайн с розовыми акцентами:

- **Фоны**: почти черные (#09090B) и темно-серые (#111116)
- **Акценты**: ярко-розовый (#FF2E88) и светло-розовый (#FF74B5)
- **Текст**: белый (#F5F5F7) и светло-серый (#A7A7B2)
- **Интерфейс**: полностью на русском языке

## История решений

Решения автоматически сохраняются в файл `data/history.json` со следующей информацией:

- Выбранный предмет
- Условие задачи
- Краткий ответ
- Полное пошаговое решение
- Путь к изображению (если есть)
- Дата и время сохранения

## Обработка ошибок

Приложение содержит полную обработку исключений:

- Глобальный обработчик необработанных ошибок
- Валидация входных данных
- Проверка целостности файлов
- Корректная обработка повреждённого JSON

## Многопоточность

Решение задач выполняется в отдельном потоке через `QThreadPool`, что предотвращает зависание интерфейса:

- Кнопка "Решить задачу" блокируется во время обработки
- Текст кнопки меняется на "Решаем…"
- После завершения кнопка восстанавливается

## Переменные окружения

Скопируйте `.env.example` в `.env` для настройки:

```bash
cp .env.example .env
```

Укажите ваш API-ключ OpenAI (когда будет подключена интеграция):

```
OPENAI_API_KEY=sk-...
```

## Сборка в .exe (Windows)

Для создания исполняемого файла Windows используйте PyInstaller:

```bash
pyinstaller --onefile --windowed --name "School AI Solver" main.py
```

Исполняемый файл появится в папке `dist/`.

## Развитие проекта

Проект готов для дальнейшего расширения:

1. **Подключение ИИ**: Замените `AIService.solve_task()` на вызов OpenAI API
2. **Распознавание текста**: Используйте tesseract или другой OCR в `ImageService`
3. **Геометрия**: Расширьте `GeometryService` для анализа и построения чертежей
4. **Новые предметы**: Добавьте в `SUBJECTS` и реализуйте специфическую логику
